                              Крестики Нолики на троих
    Программа реализована с помощью библиотеки arcade. Программа содержит класс Game унаследованный от arcade.Window.
    При создании окна ширина игровой области 900 пикселей, справа добавляется панель в 300 пикселей для отображения счётчиков ходов и подсказок. Размер клетки — 180 пикселей.
    В игре есть три игрока – Крестик(Красный), Кружок(Синий), Треугольник(Зелёный), игроки ходят по очереди.
    В правой части окна есть информация о том кто ходит, количество оставшихся ходов у каждого игрока и информация о отмене хода.
